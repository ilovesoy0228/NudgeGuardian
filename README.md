🛡️ Nudge Guardian
플랫폼의 기만적인 유도(Dark Patterns)로부터 사용자의 의사결정권을 되찾아주는 서비스

🧐 Problem Statement
현대의 플랫폼들은 자신들의 이익을 위해 사용자를 조용히 유도하는 **가짜 구조(Nudge)**를 설계합니다. 쇼핑 중 마주치는 **"품절 임박", "오늘만 이 가격"**과 같은 문구들은 사용자들을 심리적으로 압박하여 합리적인 판단을 방해하고 충동적인 결제를 유도하는 대표적인 **다크 패턴(Dark Patterns)**입니다.

Nudge Guardian은 이러한 악의적인 넛지가 감지될 때 실시간 경고 시스템을 가동하여, 사용자가 플랫폼의 의도된 유도에 휘말리지 않도록 방어막 역할을 수행합니다.

🎯 Target User
스마트 쇼퍼: 온라인 쇼핑을 자주 이용하며 합리적인 소비를 지향하는 사용자
충동 구매 방지: 광고, 마케팅, 바이럴 요소에 쉽게 이끌려 원치 않는 소비 경험이 잦은 사용자
바른 인터넷 생활 실천가: 플랫폼의 기만행위 없는 투명한 인터넷 환경을 원하는 사용자

🕹️ Control Point (핵심 가치)
"사용자의 행동과 선택에 대한 주도권 회복" 사용자는 플랫폼이 설계한 가짜 UI와 심리적 압박을 필터링함으로써, 외부의 강요가 아닌 자신의 의지에 기반한 신중한 선택을 내릴 수 있게 됩니다. 결과적으로 플랫폼에 이끌려 다니지 않는 주체적인 디지털 라이프를 통제할 수 있게 됩니다.

✨ Key Features
Real-time Detection: 기만적 키워드 실시간 탐지 및 시각적 경고 뱃지 부여

Fact Check Engine: 허위 구매 정보 및 조작된 데이터 폭로

Guardian Switch: 원클릭으로 감지 모드를 활성화하여 전/후 상태 비교 가능
